# Hướng dẫn kết nối Threads (5 phút)

## Bước 1 — Vào trang Meta Developer
- Mở trình duyệt, vào: https://developers.facebook.com
- Đăng nhập bằng Facebook của bạn

## Bước 2 — Tạo ứng dụng
- Nhấn **"Ứng dụng của tôi"** → **"Tạo ứng dụng"**
- Chọn **"Truy cập vào API Threads"** → nhấn Tiếp
- Điền tên bất kỳ (ví dụ: myapp) → nhấn **Tạo ứng dụng**

## Bước 3 — Lấy token
- Vào app vừa tạo → menu trái nhấn **Trường hợp sử dụng** → **Cài đặt**
- Kéo xuống mục **"Công cụ tạo mã người dùng"**
- Nhấn **"Tạo mã truy cập"** → đăng nhập Threads → cho phép
- Copy toàn bộ token hiện ra

## Bước 4 — Gửi token cho Admin
- Gửi token vừa copy qua Zalo cho Admin
- Admin sẽ kích hoạt tài khoản trong vòng 24h

⚠️ Lưu ý: Token có hiệu lực 60 ngày, sau đó cần lấy lại token mới.
